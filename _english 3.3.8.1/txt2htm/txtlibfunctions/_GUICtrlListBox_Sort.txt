###User Defined Function###
_GUICtrlListBox_Sort

###Description###
Re-sorts list box if it has the $LBS_SORT style

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_Sort($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
Re-sorts list box if it has the LBS_SORT style
Might be useful if you use InsertString
