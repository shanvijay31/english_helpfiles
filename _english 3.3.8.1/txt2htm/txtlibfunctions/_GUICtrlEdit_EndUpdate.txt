###User Defined Function###
_GUICtrlEdit_EndUpdate

###Description###
Enables screen repainting that was turned off with the BeginUpdate function

###Syntax###
#include <GuiEdit.au3>
_GUICtrlEdit_EndUpdate($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
